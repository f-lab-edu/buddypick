# buddypick!
#### 버디픽은 스쿠버다이빙, 프리다이빙을 즐기는 사람들이 서로 버디를 모집하고 참여할 수 있는 다이빙 커뮤니티입니다. 
#### 개인이 선택한 조건에 따라 알맞은 버디를 추천받을 수도 있으며, 인기 있는 다이빙 스팟도 확인할 수 있습니다.

## 사용기술 및 개발환경
Java21 </br>
SpringBoot 3.3 </br>
Gradle </br>
MySQL </br>
